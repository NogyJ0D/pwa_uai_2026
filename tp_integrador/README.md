# TP Final Integrador

## Estructura

- Carpeta `"tp_integrador"` contiene:
  - Carpeta `"frontend"` con la interfaz desarrollada en React + Typescript.
    - Se usó [DummyJSON](https://dummyjson.com/docs/products) para la primer entrega y el MVP.
    - Se agregaron [tailwindcss](https://tailwindcss.com/) para los estilos y [react-router-dom](https://reactrouter.com/) para la navegación.

## Iniciar

1. Para iniciar el frontend hay que acceder a la carpeta `"tp_integrador/frontend"`, ejecutar estos comandos y acceder a la URL que muestra la terminal:

   ```sh
   npm install
   npm run dev
   ```
