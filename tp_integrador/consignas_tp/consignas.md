# Consignas TP

## Primer entrega

Construir la interfaz de un e-commerce en React. La entrega debe contener un proyecto que cumpla con las siguientes cuestiones:

Mostrar productos en grilla/lista. Cada producto debe mostrar: nombre, precio, imagen
Vista individual. Debe mostrarostrar: nombre, descripción, precio, imagen
La información de dichos productos puede estar hardcodeada en el frontend.

## Segunda Entrega

Para esta segunda entrega del trabajo práctico integrador se pide el código que corresponde al backend.

Se solicita la creación de las siguientes capas: Rutas, Controladoras, Servicios y Repositorios.
Se debe codificar las APIs que completan el flujo de lo planteado en la primera entrega. Es decir, se debe crear un endpoint que consulte todos los productos y un endpoint que consulte el detalle de un producto en particular.

La entrega consiste en completar el siguiente flujo:
Frontend (primera entrega en React) -> Routes -> Controlladores -> Servicios -> Repositorios -> Base de datos
